Télécharger les vidéos des sites de streaming

Rémi Michel - le 18 janvier 2014

Introduction

Naviguez-vous sur les sites de streaming? Si oui, la suite peut vous intéresser. Il est parfois très agaçant de devoir être connecté pour pouvoir regarder un film ou une série. C'est pourquoi, j'ai décidé d'écrire ce petit article afin de vous expliquer de manière simple comment télécharger les vidéos présentes sur ces sites de streaming. Je tiens cependant à vous mettre en garde que cette petit manip ne fonctionne pas tout le temps. Je vais vous montrer une façon de faire. A vous de l'adapter en fonction du contexte dans lequel vous vous trouvez. Par ailleurs, je ne vous fais pas la morale sur le caractère légal ou non de la pratique, je pense que vous savez autant que moi quels sont les risques. Aller, passons aux choses sérieuses

Pré-requis

Le navigateur utilisé pour cette démo est Chrome. Cependant, ceci est transposable sous d'autres navigateurs dont notamment Opera ou Firefox. Chrome et Opera ont l'avantage non négligeable d'intégrer nativement une console permettant le développement. Sous Firefox, je vous recommande l'installation de Firebug

Aller sur le site de streaming

Pour la démo, je me suis rendu sur le site streamiz-filmze. Ce n'est pas le seul site proposant les vidéos en streaming bien évidemment. Une petite recherche sur google devrait aisément vous aiguiller vers ce qui se fait sur la toile en matière de streaming.

Choisissez votre film

Sur streamiz, la page est assez sommaire. On retrouve le titre du film, un court résumé et tout en bas le lecteur permettant de lancer la vidéo

Démarrage de la console

Les choses sérieuses commencent. C'est à cet instant la que vous pouvez démarrer votre console de développement. Pour cela, cliquez droit puis séléctionnez "Procéder à l'inspection de l'élément". Ceci devrait faire apparaitre la console Chrome en bas de votre écran.

Il s'agit à la base d'un outil destiné au développeur web. Pour la petite information on retrouve plusieurs onglets dont quelques-un très intéressants dont:

  • Element: Cet onglet permet l'inspection du code HTML tel qu'il est représenté sur l'écran
  • Network: Cet onglet permet de voir l'ensemble des requêtes faites. C'est dans cet onglet que nous allons passer tout notre temps
  • Console: Il s'agit de la console JavaScript

Utilisation de la console

Vous l'aurez compris, cliquez sur l'onglet "Network". Vous retrouvez ces quatre boutons:

Cliquez sur le deuxième bouton en partant de la gauche. Celui-ci aura pour effet de vider votre console. Cliquez ensuite sur le troisième bouton en partant de la gauche. Celui ci aura pour effet d'activer le tri:
Voila, vous êtes enfin prêt pour démarrer votre vidéo. Voici le nouveau contenu de votre console une fois la vidéo démarrée:
Vous remarquez une requête portant le nom "d891609fa8.240.mp4". Derrière ce nom barabare se cache quelque chose de très intéressant:
Remarquez le champs Request URL: "http://cs528508v4.vk.me/u224425259/videos/d891609fa8.240.mp4". Copiez ce lien dans un nouvel onglet de votre navigateur.
Cliquez sur "Enregistrer-sous" et le tour est joué
Regardez la constitution de l'URL "http://cs528508v4.vk.me/u224425259/videos/d891609fa8.240.mp4". Remarquez la ressource de l'URL "d891609fa8.240.mp4". Le 240 ne vous rappelle-t-il pas quelque chose? Et oui cela représente bien la qualité de la vidéo. Ainsi vous pouvez essayer de jouer avec les valeurs 360, 480, 720... Qui sait? Avec un peu de chance, la vidéo sera toujours accessible avec une meilleure qualité
